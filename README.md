## Overview

**Agent Hire Evaluators** is a web-based AI interview simulator that allows users to practice job interviews for any role. It generates interview questions, accepts text or voice answers, evaluates responses in real-time, and provides feedback along with improvement tips.  

It’s designed for candidates who want to prepare for interviews with a realistic AI-powered simulation.

---

## Features

- 🎯 Generate role-specific interview questions automatically  
- 📝 Answer questions via text input (or voice input if enabled)  
- 📊 Receive real-time evaluation including feedback, confidence score, and improvement tips  
- ✅ Progress tracking across multiple questions  
- 🏆 Summary screen displaying overall performance and top tips  
- 🌌 Modern UI with animated background and responsive design
